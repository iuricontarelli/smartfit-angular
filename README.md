## ⚙️ Como Executar

Para executar a aplicação localmente, siga os passos abaixo:

1. Clone este repositório:

```bash
  git clone https://github.com/iuricontarelli/smartfit-angular.git
```

2. Instale as dependências

```bash
  npm install
```

3. Inicie a aplicação

```bash
  ng serve
```
